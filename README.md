# 太空电梯 Space Elevator

> 连接地球与太空的未来交通

一个关于太空电梯技术的科普展示网站，介绍太空电梯的历史发展、技术原理、材料科学、面临挑战以及各国研究计划。

## 📖 项目简介

太空电梯（Space Elevator）是一种设想中的革命性太空运输系统，通过一条从地球表面延伸到地球同步轨道的缆绳，实现人员和货物在地球与太空之间的低成本运输。本网站旨在向公众科普太空电梯的相关知识，展示这一未来技术的魅力。

## 🌟 功能特色

- **响应式设计** - 适配桌面端和移动端设备
- **全屏背景视频** - 沉浸式视觉体验
- **多页面导航** - 系统化的内容组织
- **丰富的科普内容** - 涵盖历史、技术、材料、挑战等多个维度
- **交互式动画** - 平滑的页面滚动和过渡效果

## 📂 网站结构

## 🚀 快速开始

### 在线访问

访问 GitHub Pages 部署版本：
```
https://scnu2024.github.io/taikongdianti-web/
```

### 本地运行

1. 克隆仓库：
```bash
git clone git@github.com:scnu2024/taikongdianti-web.git
cd taikongdianti-web
```

2. 使用本地服务器打开：
```bash
# 使用 Python
python -m http.server 8000

# 或使用 Node.js
npx http-server
```

3. 在浏览器中访问 `http://localhost:8000`

## 📱 页面预览

- **首页** - 全屏视频背景，展示太空电梯的宏伟构想
- **历史发展** - 从康斯坦丁·齐奥尔科夫斯基到现代研究
- **技术原理** - 地球同步轨道、缆绳结构、爬升系统
- **材料科学** - 碳纳米管、石墨烯等新型材料研究
- **技术挑战** - 材料强度、空间碎片、大气层问题
- **各国计划** - 日本、美国、中国的太空电梯研究项目
- **科幻作品** - 受太空电梯启发的小说和电影
- **常见问题** - 关于太空电梯的疑问解答
